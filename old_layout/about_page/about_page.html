Description

FosterFledging empowers young adults (18–24) aging out of foster care by aggregating affordable housing, low/no-cost counseling, and trusted organizations that serve them. The site helps users find resources that fit their eligibility and needs, while also allowing community members to engage by contributing time, talent, or money.

Explanation of Disparate Data Integration

Disparate data is data that comes from various sources. In our experience, combining a broad source of data, such as the Places API (Google Maps), with something narrower like the NonProfit Explorer API (ProPublica), leads to both scope and specificity in the results. This is something that neither could do alone.
